# fitオリパ (fitoripa.com) - プロジェクト仕様書

## プロジェクト概要
フィットネス × オリパのオンラインサービス。
- サービス名: fitオリパ
- ドメイン: fitoripa.com
- コンセプト: フィットネス×オリパ・爽やか・健康的

## 技術スタック
- フロントエンド: Next.js (App Router) / TypeScript
- DB: Supabase (PostgreSQL)
- 認証: Supabase Auth (Google認証含む)
- 決済: Stripe (クレカ・コンビニ・銀行振込)
- デプロイ: Vercel
- バージョン管理: GitHub

## 実装済み機能
- ログイン・会員登録（Google認証）
- ガチャ機能（1口・複数口・天井カウンター）
- 演出動画（賞ごとに複数動画・ランダム再生）
- マイページ・ポイント残高
- 獲得商品一覧・売却・コイン交換
- ガチャ履歴・当選報告
- 発送申請
- ポイント購入（クレカ・コンビニ・銀行振込）
- 管理画面（商品・オリパ・バナー・演出動画・顧客・注文・ポイント管理）

## 賞のレアリティ
- S賞: 最高レア・虹色演出
- A賞: 高レア・パーティクル爆発
- B賞: 中レア・青いグロー
- C賞: 通常・シンプル演出
